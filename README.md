# Student-Manager-Java
Basic student management system meant as a Java version of the other student management system repo.

Has a handful of differences as per project requirements.

## Features of this Program:
* Parses and stores student data from a given table, defined in Student_Main.java
* Adds, removes, and displays student information
* Ensures correct email address formatting from email addresses provided in the student data table
* Functions for showing various kinds of student data, particularly average grade
